### **Bold Text**

```markdown
**Bold Text** or __Bold Text__ or <strong></strong> or <b></b>
```

### *Italic Text*

 ```markdown

 ```

### Unordered List 
- Apple
- Mango
- Banana
- Kiwi

```markdown
- Apple |
- Mango |
- Banana |
- Kiwi |

or 

* Apple
* Mango
* Banana
* Kiwi

<ul>
 <li>Apple</li>
 <li>Mango</li>
 <li>Banana</li>
 <li>Kiwi</li>
</ul>
```

### Ordered List 

1. Apple
2. Samsung
3. OnePlus
4. Mi
5. Realme

```markdown
1. Apple
2. Samsung
3. OnePlus
4. Mi
5. Realme

or 

<ol type='1'>
 <li>Apple</li>
 <li>Samsung</li>
 <li>OnePlus</li>
 <li>Mi</li>
 <li>Realme</li>
</ol>
```
